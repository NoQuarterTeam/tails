<!-- usage -->

```sh-session

$ tails (-v|--version|version)

$ tails --help [COMMAND]

  $ tails COMMAND
...
```

# Commands

<!-- commands -->

## `tails scaffold`

```
USAGE
  $ tails scaffold

OPTIONS
  -f, --force
  -h, --help       show CLI help
  -n, --name=name  name to print

EXAMPLE
  $ tails scaffold
```

## `tails help [COMMAND]`

display help for tails

```
USAGE
  $ tails help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```
